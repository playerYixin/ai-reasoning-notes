# 全书思维导图

> **来源**: [推理王国](https://github.com/datawhalechina/reasoning-kingdom)  
> **覆盖范围**: 卷一 Ch1–13 + 番外、卷二导读 + Ch14–23

```
推理 = 对抗熵增 (Ch1)
  ├── 符号推理 → 知识瓶颈 (Ch2)
  │     └── 词向量 → 语义几何 (Ch3)
  │           └── 流形假设 → 高维结构 (Ch4)
  ├── 统计学习 → 相关性陷阱 (Ch5)
  │     └── 因果边界 → do-calculus (Ch6, Ch18)
  ├── 计算复杂度 → P/NP 边界 (Ch7, Ch19)
  │     ├── 启发式 → A* 契约 (Ch8, Ch20)
  │     └── 搜索 → MCTS (Ch10)
  ├── 神经网络架构
  │     ├── Transformer → 注意力机制 (Ch9)
  │     ├── 隐式推理 → CoT (Ch12)
  │     └── 效能 → Collins 优化器 (Ch11)
  ├── 推理边界
  │     ├── 不完备性 → 永霖公式 (Ch13)
  │     └── 形式系统地基 (Ch14–Ch22)
  │           ├── 一致性/完备性 → 哥德尔 (Ch15)
  │           ├── 线性逻辑 → 资源 (Ch16)
  │           ├── 概率逻辑 → Cox 定理 (Ch17)
  │           ├── 学习理论 → MDL (Ch21)
  │           └── Curry-Howard → 证明=程序 (Ch22)
  └── 动力学稳定性 → 李雅普诺夫 (Ch23)
        └── 范畴论视角 → 伴随函子 (Ch24)
```
